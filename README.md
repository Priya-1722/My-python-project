# Data Analytics & Automation Pipeline

A lightweight Python backend data engine that automates the ingestion, cleaning, and prioritization of raw e-commerce feedback datasets.

## 🚀 Core Features
* **Data Cleaning:** Automatically filters and standardizes raw market data inputs.
* **Automated Tagging:** Uses logic filters to segregate critical customer complaints from positive feedback.
* **KPI Generation:** Dynamically computes a live corporate Customer Satisfaction Index (CSI) percentage.

## 🛠️ Tech Stack
* **Language:** Python 3
* **Tools:** GitHub Version Control
